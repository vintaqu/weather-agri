# Meteo España
Aplicación web para visualizar datos meteorológicos de España con mapa interactivo.

## Backend
- Node.js + Express
- Endpoint `/api/weather` que obtiene datos de Open-Meteo

## Frontend
- HTML, CSS, JavaScript
- Leaflet.js para mapa interactivo

## Instalación
```bash
cd backend
npm install
npm start
```
En otra terminal:
```bash
cd frontend
# abrir index.html en el navegador
```
